### BUG_REPORT
Title: StringUtils.join throws NPE when toString returns null for one of objects in collection

Description:
Try

 
StringUtils.join(new Object[]{
        new Object() {
          @Override
          public String toString() {
            return null;
          }
        }
    }, ',');


ToString should probably never return null, but it does in javax.mail.internet.InternetAddress
